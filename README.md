## Normal Installation

    npm i -g linlang

## Manual Installation

`git clone` this repo, `cd` into it, `npm install`, and `npm ln`.

## Language Specs

CLI Usage:

    lin program.lin

For more options, run `lin -h`.

More information can be found on the [wiki](https://github.com/molarmanful/lin/wiki).
